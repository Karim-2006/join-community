A simple React + TypeScript application for collecting signups to join a WhatsApp community. Built with Vite, Tailwind CSS, and shadcn-ui.

## Tech Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- shadcn-ui

## Getting Started

Prerequisites:
- Node.js 18+ and npm

Install dependencies and start the dev server:

```sh
npm install
npm run dev
```

Visit the app at the printed local URL.

## Build

Create a production build:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Notes

- This project has no third-party branding.
- Open Graph/Twitter image tags are intentionally disabled to avoid branded images.
